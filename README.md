# Sistema de Gestão para Marcenaria (CMD)

Sistema de gestão desenvolvido em **Python**, executado via **linha de comando (CMD)**, voltado para o controle de **produtos, categorias, fornecedores e vendas** de uma marcenaria.

O projeto segue o padrão **MVC (Model–View–Controller)**, visando organização, escalabilidade e facilidade de manutenção.

---

## Funcionalidades

 Gestão Completa (CRUD)
- **Produtos**: cadastro, edição, remoção e listagem
- **Categorias**: cadastro, edição, remoção e listagem
- **Fornecedores**: cadastro, edição, remoção e listagem
- **Clientes**: gerenciamento de clientes e histórico de compras
- **Funcionários**: cadastro e controle de usuários internos
- **Vendas**: registro, consulta e detalhamento de vendas

  Sistema de Caixa
- **Caixa dinâmico**, com cálculo automático de valores

  Relatórios Gerenciais
- Relatório de **vendas por período**
- Relatório de **produtos mais vendidos**
- Relatório de **clientes que mais compraram**
- Apoio à tomada de decisão do negócio

---

## Executar Projeto

- Comando **python -m Produtos.View**

---
  ## Arquitetura e Boas Práticas
- Arquitetura **MVC (Model–View–Controller)**

- Commits atômicos no padrão **Conventional Commits**
